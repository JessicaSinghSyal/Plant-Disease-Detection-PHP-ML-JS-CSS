# Plant-Disease-Detection-PHP-ML-JS-CSS

Plant Disease Detection and Treatment Recommendation System
This project is an ML-based application utilizing a Convolutional Neural Network (CNN) to detect plant diseases from images. The frontend, built with React and JavaScript, provides an intuitive interface for users to upload photos and receive diagnostic results. The backend, developed with PHP, handles image processing, model inference, and data management. Upon detecting a disease, the system offers detailed information about the disease, suggests treatments, recommends fertilizers, and provides purchase links for necessary products.
